# BoardRate

## Introduction

BoardRate is an mobile Android review application for board game enthusiasts that can crowdsource user feedback in a simple, easy-to-navigate way.
Much of the design layouts of exisiting board game websites are extremely archaic and cluttered and would require new users a decent amount of time to become familiar with the interface which can be off putting to those who donâ€™t necessarily have that kind of desire or time. BoardRate hopes to cater to the causal board game players with simple, yet powerful UI design choices as well as any easy to understand ratings and review system. 

## User Personas:
1. Expert players
2. Novice players
3. Cafe owners/ Gameshop owners

## User Tasks:
1. Catch the Trend
2. Search for Games
3. Write Reviews & Keep a Journal
